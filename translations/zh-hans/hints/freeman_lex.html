您只需将您的代码封装为函数表达式，并将所有之前的提示连接在一起。

这就是整个解决方案：

from collections.abc import Iterable


def duplicate_zeros(donuts: list[int]) -> Iterable[int]:
    
    res = []
    for don in donuts:
        res.append(don)
        if not don:
            res.append(don)
    return res

使用yield from 可以避免创建一个新的结果列表，它将小列表中的每个元素 "发送 "到函数之外。如果元素等于0 ，它将发送其中的两个。 

from collections.abc import Iterable


def duplicate_zeros(donuts: list[int]) -> Iterable[int]:

    for don in donuts:
        yield from [don] * (2-bool(don))

递归版本：返回空列表（如果没有初始列表）或列表中的第一个元素（如果有必要，返回两次），然后重复返回较小的初始列表。将所有这些列表连接在一起。 

from collections.abc import Iterable


def duplicate_zeros(donuts: list[int]) -> Iterable[int]:

    if not donuts:

        return []

    return [donuts[0]] + [0] * (donuts[0] == 0) + duplicate_zeros(donuts[1:])
