So, how my final solution should look like?

You just need to wrap your code into function expression and join all previous hints together.

Here is the whole solution:

from typing import Iterable


def duplicate_zeros(donuts: list[int]) -> Iterable[int]:
    
    res = []
    for don in donuts:
        res.append(don)
        if not don:
            res.append(don)
    return res

Using yield from allows us to avoid creating a new resulted list, it "sends" each element from the small list outside the function. If the element equals 0, it sends two of them. 

from typing import Iterable


def duplicate_zeros(donuts: list[int]) -> Iterable[int]:

    for don in donuts:
        yield from [don] * (2-bool(don))

Recursive version: return an empty list (if there is no initial list) or first element in list (twice if necessary), than repeat for the smaller initial list. Join all these lists together. 

from typing import Iterable


def duplicate_zeros(donuts: list[int]) -> Iterable[int]:

    if not donuts:

        return []

    return [donuts[0]] + [0] * (donuts[0] == 0) + duplicate_zeros(donuts[1:])
